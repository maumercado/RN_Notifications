# RN Notifications

A simple React Native exercise for implementing and testing push notifications. Practice project for learning notification APIs and scheduled notifications.

## Features

Send local push notifications
Schedule notifications for specific times
Handle notification permissions
Test notification delivery on device
Demonstrate notification best practices

## Requirements

Node.js (specified in .nvmrc)
React Native development environment
iOS Simulator or Android Emulator
Physical device recommended for full testing

## Installation

Clone the repository
Install dependencies:
```
npm install
```

Start the development server:
```
npm start
```

Run on iOS or Android following React Native CLI instructions

## Technologies

React Native
JavaScript
Expo Notifications (if applicable)
Native notification APIs
